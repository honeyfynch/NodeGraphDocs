# Foundation recontribution scaffold (reference only)

These files are **not** part of a runnable package in this repository. They mirror the layout expected under:

`modules/foundation/src/Components/<ComponentName>/`

in the [Foundation](https://github.com/Roblox/foundation) repo, following the [View](https://github.com/Roblox/foundation/tree/main/modules/foundation/src/Components/View) component pattern.

**Instructions:** Copy `GraphNodeShell/` into Foundation, fix `FindFirstAncestor` paths, implement real props and styling from [foundation-graph-node-recontribution.md](../foundation-graph-node-recontribution.md), register the component in `modules/foundation/src/init.lua`, add FFlags and `.changes/` per AGENTS.
